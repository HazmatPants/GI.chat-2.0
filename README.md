# GIchat Client 2.0
Used to connect to [GIchat servers](https://github.com/HazmatPants/GIchat-server)

A lightweight chat client for GIchat. Built with PyQt5, WebSockets and Flask (HTTP).

# Features
- Customizable sounds
- Markdown support
- HTML formatting support
